# Blog Platform

A full-stack blog platform built with Next.js, Supabase, and Tailwind CSS.

## Getting Started

1.  Clone the repository.
2.  Install dependencies: `npm install`
3.  Set up your Supabase project and copy the `.env.local.example` file to `.env.local`. Fill in the environment variables.
4.  Start the Supabase database: `supabase start`
5.  Run the development server: `npm run dev`

## Deployment

Deploy to Vercel directly from your repository.
